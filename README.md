# Number Guessing Game (CMPSC 132)

## Project Description
This is a terminal-based number guessing game built using Python.  
The program generates a random number within a selected range, and the player must guess the number based on feedback.

## Features
- Difficulty levels (Easy, Medium, Hard)
- Random number generation using Python
- Input validation (prevents invalid inputs)
- Replay option to play multiple rounds

## How the Game Works
1. The user selects a difficulty level:
   - Easy: 1–50
   - Medium: 1–100
   - Hard: 1–200  
2. The program generates a random number within the chosen range.
3. The user enters guesses.
4. The program gives feedback:
   - "Too low"
   - "Too high"
5. The game continues until the correct number is guessed.
6. The number of attempts is displayed.
7. The user can choose to play again.

## Technologies Used
- Python

## How to Run the Program
1. Make sure Python is installed.
2. Download or clone this repository.
3. Run the program using:

python number_guessing_game.py
